# 🧠 Deep Learning Notes

A structured collection of traditional Deep Learning notes, neural network implementations, and practical examples created while learning core Deep Learning concepts and architectures.

This repository focuses on building strong foundations in Deep Learning using mathematical intuition, coding implementations, and model training workflows.

The primary goal of this repository is to cover traditional Deep Learning models and concepts from beginner to advanced level.

> Note: Modern GenAI, LLMs, RAG systems, AI agents, and transformer-based applications will be maintained separately in the `genai-notes` repository.
